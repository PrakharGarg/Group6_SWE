Data

  • Source 1: Untapped API
    • Phase 1: Since we are serving up static pages and don't have a database to work with, we manually looked up breweries, locations, beers, and breweries. We manually imported this data into our project.
    • Phase 2: Using Untapped's API, we first queried all of the venues. With the Venues came the list of beers and breweries that they served. Using this list of breweries, we then queried all of their beers.
  • Source 2: Wikipedia API
    • Phase 1: Since we are serving up static pages and don't have a database to work with, we manually looked up breweries, locations, beers, and breweries. We manually imported this data into our project.
    • Phase 2: Using Wikipedia's API, we created a script to go through the U.S States page and query each of the states listed. From there, we massaged the json down to only the fields that we needed.
